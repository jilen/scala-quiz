> 开始使用

* 下载[sbt](http://www.scala-sbt.org/download.html)
* 安装sbt。确保`sbt`命令在终端中可以被调用
* 下载一些[依赖](http://pan.baidu.com/s/1sjJWyD3)。因为GFW原因，sbt第一次加载非常缓慢，建议下载前面的文件。并解压到`$HOME/.ivy2`(`$HOME`为用户目录)
* Fork该项目，然后`git clone`fork的项目到本地进行开发
* cd到项目目录，运行sbt
* 编写/测试
